# Digital Resilience & Climate Adaptive Capacity in the Arab Region

### Project Overview
This project explores the correlation between digital infrastructure (Internet Penetration & Secure Servers) and climate change adaptive capacity (ND-GAIN Readiness) across the 22 Arab States.

### Key Findings
- **Statistically Significant:** Found a positive correlation (R² = 0.49, p-value = 0.003) between digital connectivity and climate resilience.
- **Regional Divide:** Visualized a clear gap between high-connectivity leaders and vulnerable states in the Levant and Horn of Africa.

### Technical Workflow
1. **Data Wrangling:** Python (Pandas) script to standardize wide-format CSVs from World Bank and ND-GAIN.
2. **Econometrics:** Calculated linear regressions to validate the tech-climate nexus.
3. **Visualization:** Interactive Tableau Dashboard.
